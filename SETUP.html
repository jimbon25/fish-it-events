üìã Step 1: Siapkan GitHub Repository

Buat repository baru untuk menyimpan file events.json

  1. Pergi ke GitHub > Create New Repository
  2. Nama repo: fish-it-events
  3. Pilih Public (agar bisa di-fetch)
  4. Add README.md (optional)
  5. Create repository
git clone https://github.com/YOUR_USERNAME/fish-it-events.git cd fish-it-events # Copy project files ke folder ini

üîë Step 2: Generate GitHub Personal Access Token

Token ini digunakan untuk update events.json via API

  1. Pergi ke GitHub > Settings > Developer settings > Personal access tokens
  2. Klik "Generate new token"
  3. Set note: "Fish It Landing Page"
  4. Pilih scope: repo (full control of private repositories)
  5. Klik "Generate token"
  6. COPY token - jangan lupa! (hanya akan ditampilkan 1x)

  • Jangan share token ke siapapun
  • Jangan commit ke Git

üìù Step 3: Update Konfigurasi

Update GitHub username di file konfigurasi

Edit file: script.js
// Line ~13 EVENTS_URL: 'https://raw.githubusercontent.com/YOUR_USERNAME/fish-it-events/main/events.json' // Ganti YOUR_USERNAME dengan username GitHub Anda
Edit file: admin.js
// Line ~6-8 GITHUB_USERNAME: 'YOUR_USERNAME', GITHUB_REPO: 'fish-it-events', GITHUB_BRANCH: 'main', // Ganti YOUR_USERNAME dengan username GitHub Anda

üöÄ Step 4: Deploy ke Vercel

Hosting gratis dan cepat untuk static site

  1. Pergi ke vercel.com
  2. Sign up / Login (bisa pakai GitHub account)
  3. Klik "New Project"
  4. Import repository fish-it-events
  5. Konfigurasi:
    • Framework: "Other" (static)
    • Build Command: (kosongkan)
    • Output Directory: (kosongkan)
  6. Klik "Deploy"

üîê Step 5: Setup GitHub Token di Vercel

Agar admin dashboard bisa save event ke GitHub

  1. Buka project di Vercel dashboard
  2. Pergi ke Settings
  3. Pilih Environment Variables
  4. Klik "Add New"
  5. Fill form:
    • Name: GITHUB_TOKEN
    • Value: [Paste token dari Step 2]
  6. Select all environments (Production, Preview, Development)
  7. Klik "Save"
  8. Redeploy project
‚úÖ Selesai! Environment variable sudah tersimpan. Redeploy otomatis akan start.

üÜò Troubleshooting

Events tidak muncul di landing page?
  • Check console (F12) untuk error messages
  • Verify GitHub URL di script.js
  • Pastikan events.json sudah di-push ke GitHub
  • Cek GitHub rate limiting (60 req/jam tanpa auth)
Admin tidak bisa save event?
  • Verify GitHub token di Vercel environment variables
  • Check browser console (F12) untuk error
  • Pastikan repository bisa diakses
  • Verify token permission di GitHub settings
CORS Error?
  • GitHub raw content URL support CORS - tidak perlu setup
  • Verify URL format benar
  • Check console untuk exact error message
